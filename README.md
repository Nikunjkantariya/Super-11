🏏 **SUPER 11 - T20 World Cup 2022's Best Playing XI Analytics**

📊 **Project Overview** :

This project analyzes **T20 World Cup 2022 cricket data** to identify the **Best Playing XI team** using data-driven insights.

The project follows a complete **data analytics workflow** including:

- Data Collection
- Data Cleaning
- Data Transformation
- Data Modeling
- Data Visualization
- Insight Generation

The final output is an **interactive Power BI dashboard** that helps evaluate player performance and build an optimal cricket team based on statistical analysis.

🎯 **Problem Statement** :

Selecting the best cricket team often depends on subjective opinions. This project uses **data analytics techniques** to objectively analyze player performance and identify the most suitable **Playing XI combination**.

The dashboard evaluates players based on key performance metrics such as:

- Batting Strike Rate
- Batting Average
- Boundary Percentage
- Economy Rate
- Total Wickets
- Bowling Strike Rate

Using these metrics, the dashboard helps build a **balanced and high-performing cricket team**.

📂 **Dataset** :

The dataset is based on **T20 World Cup 2022 player statistics** collected from **ESPN Cricinfo**.

Data includes:

- Player profiles
- Batting statistics
- Bowling statistics
- Match performance metrics

🔄 **Data Analytics Workflow** :

Cricket statistics were collected from **ESPN Cricinfo** and stored in structured datasets for analysis.

2️⃣ **Data Cleaning**

Using **Pandas**, the dataset was cleaned by:

- Removing missing values
- Formatting columns
- Filtering relevant statistics

3️⃣ **Data Transformation**

The data was transformed into structured tables including:

- Player Profiles
- Batting Statistics
- Bowling Statistics

4️⃣ **Data Modeling**

Relationships between datasets were created in **Power BI** to enable effective analysis.

5️⃣ **Dashboard Development**

Interactive dashboards were created to analyze:

- Top Batters
- Best Bowlers
- All-rounders
- Finishers
- Role-based Player Comparison
